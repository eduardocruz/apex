0G iNFT explorer

Read-only viewer for the Agentic ID (ERC-7857) registry on the 0G Galileo Testnet.

What is this? 0G is an EVM L1 ("Galileo Testnet", chain id 16602) running an experimental NFT standard called ERC-7857 — "Agentic ID". The idea: an AI agent's identity (its name, model, capabilities, system prompt) can be minted as a transferable, cloneable NFT.

The privacy twist: the agent's actual data is not stored on-chain. Only the field names ("agent_name", "model", …) are public. Each value is committed as a bytes32 hash. The real value lives off-chain (in 0G Storage, encrypted) and the hash on-chain is the cryptographic anchor.

What this page does: reads the registry contract directly from the chain. No wallet, no signing, no testnet tokens. You see what's actually committed when someone "registers an agent".
